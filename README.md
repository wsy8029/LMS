  ##### 행동 로그 지표
    1.행동 빈도
      - 북마크 및 코멘트 열람 빈도 -> 학습 중 앞 내용을 다시 열람하는 빈도
      - 북마크 설정 및 확인 빈도 -> 북마크 설정 및 확인 빈도
      - 코멘트 작성 및 확인 빈도 -> 코멘트 작성 가능 여부 확인?
      - 외부 환경 조정 빈도 -> 외부 환경?
      - 주석 필터링 빈도 ->
      - 일시정지 빈도 -> 
      - 재생 빈도 ->
      - 슬라이드 조정 빈도 ->
      - 동료 학습자 코멘트 답글 작성 빈도 ->
      - 좋아요 누른 빈도
